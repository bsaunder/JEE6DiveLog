The following resources are part of this group:

  • /user
  • /user/find
  • /user/register

/user

Mount Point: /rest/user

GET

Gets all Users.

  • Status 200: Users Found.
  • Status 400: No Users Found.

Response Body

element: (custom)

List of Users

/user/find

Mount Point: /rest/user/find

POST

Gets the User Specified by the Given Criteria. If not Criteria Specified, All Results will be returned.

  • Status 200: Results Found.
  • Status 400: Results not found.

Request Body

element: userAccount

Criteria to search for

Response Body

element: (custom)

Found users

/user/register

Mount Point: /rest/user/register

POST

Registers a User.

  • Status 200: Successful Registration.
  • Status 400: Error with the Request. Most likely caused by an invalid UserAccount.

Request Body

element: userAccount

User to Register

Response Body

element: userAccount

Registered User
