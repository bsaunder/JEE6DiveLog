/user/find

Mount Point: /rest/user/find

GET

Gets the User Specified by the Given Username.

  • Status 200: User Found.
  • Status 404: User was not found.

Parameters

name description type default
userName Username to serach for query

Response Body

Found user

Mount Point: /rest/user/register
