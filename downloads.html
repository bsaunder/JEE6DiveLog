C Client Library

Created Jul 31, 2012 6:56:44 PM

Introduction

The C module generates the source code for the ANSI-C-compatible data structures and (de)serialization functions that can be used in conjunction with libxml2 to (de)serialize the REST resources as they are represented as XML data.

The generated C source code depends on the XML Reader API and the XML Writer API as well as the <time.h>, <string.h>, and <stdlib.h> C standard libraries.

REST XML Example

#include <novaglobalapi.c> //... xmlTextWriterPtr writer = ...; //set up the writer to the url. novaglobalapi_ns0_userAccount *request_element = ...; xmlTextReaderPtr reader = ...; //set up the reader to the url. novaglobalapi_ns0_userAccount *response_element = ...; //set up the novaglobalapi_ns0_userAccount... xml_write_novaglobalapi_ns0_userAccount(writer, request_element); response_element = xml_read_novaglobalapi_ns0_userAccount(reader); //handle the response as needed... //free the novaglobalapi_ns0_userAccount free_novaglobalapi_ns0_userAccount(request_element); //free the novaglobalapi_ns0_userAccount free_novaglobalapi_ns0_userAccount(response_element);

Java Client Library

Created Jul 31, 2012 6:56:45 PM

Introduction

The Java client-side library is used to access the Web service API for this application.

The JAX-WS client-side library is used to provide the set of Java objects that can be serialized to/from XML using JAXB. This is useful for accessing the REST endpoints that are published by this application.

REST Example (Raw JAXB)

java.net.URL url = new java.net.URL(baseURL + "/user/register"); JAXBContext context = JAXBContext.newInstance( UserAccount.class, UserAccount.class ); java.net.URLConnection connection = url.openConnection(); connection.setDoOutput(true); connection.connect(); Unmarshaller unmarshaller = context.createUnmarshaller(); Marshaller marshaller = context.createMarshaller(); marshaller.marshal(userAccount, connection.getOutputStream()); UserAccount result = (UserAccount) unmarshaller.unmarshal( connection.getInputStream() ); //handle the result as needed...

REST Example (Jersey client)

com.sun.jersey.api.client.Client client = com.sun.jersey.api.client.Client.create(); UserAccount result = client.resource(baseUrl + "/user/register") .entity(userAccount) .post(UserAccount.class); //handle the result as needed...

Objective C Client Library

Created Jul 31, 2012 6:56:44 PM

Introduction

The Objective C module generates the source code for the Objective C classes and (de)serialization functions that can be used in conjunction with libxml2 to (de)serialize the REST resources as they are represented as XML data.

The generated Objective C source code depends on the XML Reader API and the XML Writer API as well as the base OpenStep foundation classes.

REST XML Example

#import <novaglobalapi.h> //... NOVAGLOBALAPINS0UserAccount *requestElement = [[NOVAGLOBALAPINS0UserAccount alloc] init]; NSData *requestData; //data holding the XML for the request. NOVAGLOBALAPINS0UserAccount *responseElement; NSData *responseData; //data holding the XML from the response. NSURL *baseURL = ...; //the base url including the host and subpath. NSURL *url = [NSURL URLWithString: @"/user/register" relativeToURL: baseURL]; NSMutableURLRequest *request = [[NSMutableURLRequest alloc] initWithURL:url]; NSURLResponse *response = nil; NSError *error = NULL; [request setHTTPMethod: @"POST"]; [request setValue:@"application/xml" forHTTPHeaderField:@"Content-Type"]; //set up the NOVAGLOBALAPINS0UserAccount... requestData = [requestElement writeToXML]; [request setHTTPBody: requestData]; //this example uses a synchronous request, //but you'll probably want to use an asynchronous call responseData = [NSURLConnection sendSynchronousRequest:request returningResponse:&response error:&error]; NOVAGLOBALAPINS0UserAccount *responseElement = [NOVAGLOBALAPINS0UserAccount readFromXML: responseData]; [responseElement retain]; //handle the response as needed...
