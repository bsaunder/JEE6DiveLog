JEE6 Dive Log

Environment Setup: 1) Install MySQL 2) Create Database & User 3) Install JBoss AS7 or Red Hat EAP6 4) Configure Datasource in standalone.xml jdbc:mysql://127.10.101.1:3306/jee6divelogmysqljee6divelogjee6divelog1com.mysql.jdbc.jdbc2.optional.MysqlXADataSource

Build & Test: 1) mvn clean install

Deploy: 1) place .war file in deployments directory

Technology Used: Unit tests are written using JUnit 4, Mockito 1.9, and PowerMock 1.4. Integration tests are written using Arquillian 1.2.
